This is project is done for roadmap.sh task:
https://roadmap.sh/projects/task-tracker

Start: `npm run start` at root

## Commands
add - Adds a new db entry into the JSON file 
update - Updates an existing entry in the database
delete - Delete an existing entry in the databse
mark-in-progress - Change existing task status to in-progress
mark-done - Change existing task status to done
list - List all tasks
list done - List all tasks with status done
list todo - List all tasks with status todo
list in-progress - List all task with status in progress
